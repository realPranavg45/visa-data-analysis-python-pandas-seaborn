# 🌍 Visa Data Analysis using Python

## 🚀 Overview

This project focuses on analyzing visa application data to uncover patterns, trends, and key factors influencing visa approvals and rejections.

Using Python-based data analysis techniques, raw data is transformed into meaningful insights through structured exploration and visualization.

---

## 🎯 Objective

* Understand visa approval vs rejection trends
* Identify key factors affecting visa decisions
* Analyze patterns across different categories
* Derive actionable insights from data

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 🔍 Analysis Performed

* Data cleaning and preprocessing
* Handling missing values
* Exploratory Data Analysis (EDA)
* Distribution analysis
* Category-wise comparisons
* Correlation analysis (heatmap)
* Outlier detection using boxplots

---

## 📊 Key Insights

* Visa approvals are higher compared to rejections in the dataset
* Higher salary is associated with higher chances of approval
* Certain categories show noticeable rejection patterns
* Experience and job-related factors influence outcomes
* Data shows skewed distributions in key variables

---

## 💡 Conclusion

The analysis highlights that visa approval is influenced by multiple factors such as salary, job category, and experience level.

These insights can help in understanding application trends and making data-driven decisions.
